# ED-II
# 🔍 Sistema de Consulta Eficiente com Tabela Hash e Filtro de Bloom

---

## 📋 Visão Geral

Este projeto implementa um **sistema eficiente de consulta de dados** capaz de armazenar e buscar grandes volumes de registros de usuários (milhões de dados) com performance otimizada.

Utilizamos duas estruturas de dados fundamentais:
- **Tabela Hash**: Para armazenamento e busca rápida de registros em O(1) tempo médio
- **Filtro de Bloom**: Para verificação probabilística ultra-rápida de existência de elementos

### 🎯 Objetivo

Desenvolver uma solução prática que demonstre como estruturas de dados avançadas podem resolver problemas reais de recuperação eficiente de informações em conjuntos de dados massivos.
